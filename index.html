Set up BrewPilot
Two minutes. Pick what you do, then connect a sheet if you want your data saved.
1What do you want to do?
What do you brew?
What gear do you have?
Which grinders do you own?
Set up your Google Sheet
Your data lives in your own Google Sheet. Three steps, about two minutes.
2Copy the template
The template already has the code inside. A blank sheet will not work.
Open the template
In the menu: File > Make a copy. The copy is yours, in your Drive.
3Open the code editor
Extensions › Apps Script
A new tab opens. This is the code that will save your brews.
4Publish it as a web app
1. Deploy › New deployment
2. Click the gear icon, choose Web app.
3. Execute as: Me
4. Who has access: Anyone <-- this one matters
5. Press Deploy.
!Google will warn you. This is normal.
It says Google hasn't verified this app. That is because the app is yours, written by you, not published to the store.
Press Advanced, then Go to (unsafe), then Allow.
5Copy the link and paste it here
Copy the Web app URL. It ends in /exec.
